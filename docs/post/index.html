06-06-2017 Elm Elm-lang

Le but de cette série est de réaliser un petit programme en utilisant Elm pour gérer l’affichage (frontend). Qu’est-ce que Elm ? Elm est un langage de programmation fonctionnel qui permet de créer des interfaces graphiques pour le web ainsi que des jeux. Il compile le code en JavaScript. – Source : Wikipedia Elm est un langage fonctionnel comme OCaml, F#, Haskell, Elixir, Scala, Clojure, etc. Sa syntaxe et ses concepts sont donc très différents des langages impératifs comme le JavaScript et des dérivés comme TypeScript ou PureScript.
